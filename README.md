# eXobe Assessment

Initial app foundation for the eXobe Phase 2 technical assessment.

## What exists

- Next.js App Router
- React and TypeScript
- Tailwind CSS
- shadcn/ui initialized with the Button component
- ESLint via `pnpm lint`
- Basic branded starter screen

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.
